# Competing LLM Frontend

This is the frontend for the Competing LLM application, built with React, Vite, Tailwind CSS, and Shadcn UI.

## Prerequisites

- Node.js (v18+)
- Backend server running on `http://localhost:8000`

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`.

## Features

- Select multiple LLMs to compare.
- Send prompts to selected models.
- View responses side-by-side in a grid layout.
